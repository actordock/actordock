# v0.1.0

**Status:** released

## Target

**Pilot + 60% E2B REST (ops + fields)** — Agent workloads on non-Kind clusters; every shipped route has **complete** OpenAPI schemas.

| Before | After |
|--------|-------|
| Dev Kind stack; ~13% ops, ~20% fields | Helm; **≥33 ops + ≥60% fields** (full-field rule) |

Not production-grade (no HA, template builds, admin API).

## Goal

```python
s = Sandbox.create(template="base")  # secure=True
s.files.write("/tmp/x", "data")
assert s.files.read("/tmp/x") == "data"
s.commands.run("cat /tmp/x").stdout.strip() == "data"
s.kill()
```

```bash
helm install actordock ./charts/actordock-stack -n actordock --create-namespace
```

Verify: [e2b-rest-parity.md](e2b-rest-parity.md) — **33 operations `done`**, **≥60% fields**, no `partial` rows left for in-scope routes.

## In scope

### Pilot (non-REST)

| Area | Deliverable |
|------|-------------|
| envd | Filesystem RPC: read, write, list |
| Platform | `secure=true`; `envd_access_token` on create |
| Router | Forward auth headers to envd |
| Deploy | Helm `actordock-stack`; Substrate pin in chart/lock |
| Docs | Install runbook, env vars, troubleshooting |

### REST (+6 ops, all fields)

| Method | Path | Schemas (full) |
|--------|------|----------------|
| POST | `/templates` | `NewTemplate` → `Template` |
| PATCH | `/templates/{templateID}` | `TemplateUpdateRequest` → `TemplateUpdateResponse` |
| GET | `/api-keys` | `TeamAPIKey[]` |
| POST | `/api-keys` | `NewTeamAPIKey` → `CreatedTeamAPIKey` |
| POST | `/access-tokens` | `NewAccessToken` → `CreatedAccessToken` |
| DELETE | `/access-tokens/{accessTokenID}` | — |

### Sandbox field completion (v0.1.0 closes `partial`)

- `POST /sandboxes`: all 11 `NewSandbox` + 8 `Sandbox` response fields
- `GET /sandboxes/{id}`: all 18 `SandboxDetail` fields
- `GET /sandboxes`, `GET /v2/sandboxes`: all 13 `ListedSandbox` fields per item

## Out of scope

- Admin, nodes, `GET /teams` and team metrics
- Full multi-tenant auth, HA, SLA
- Remaining 23 REST ops (see parity tracker)
