# v0.0.10

**Status:** planned

## Target

**Templates (read)** — List and inspect templates; resolve alias/tags.

| Before | After |
|--------|-------|
| Only hardcoded `base` | E2B template GET routes work for pre-provisioned templates |

## Goal

```python
# SDK Template.list / get / resolve alias — no 404
t = Template.get("base")  # or alias lookup
```

## In scope

| Area | Deliverable |
|------|-------------|
| Platform | `GET /templates`, `GET /templates/{id}` |
| Platform | `GET /templates/aliases/{alias}`, `GET /templates/{id}/tags` |
| Platform | `GET /templates/{id}/files/{hash}` (stub or static for `base`) |
| Store | Template catalog synced from cluster ActorTemplates (minimal) |

## Out of scope

- Template build (POST v2/v3, build logs/status)
- DELETE/PATCH template (PATCH deferred to v0.1.0)
- Custom user-built templates

## API

| Method | Path |
|--------|------|
| GET | `/templates` |
| GET | `/templates/{templateID}` |
| GET | `/templates/aliases/{alias}` |
| GET | `/templates/{templateID}/tags` |
| GET | `/templates/{templateID}/files/{hash}` |
