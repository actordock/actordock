# v0.1.0

**Status:** planned

## Target

**Pilot** — Real agent workloads on a non-Kind cluster with files and auth.

| Before | After |
|--------|-------|
| Dev-only Kind stack | Helm on any K8s; read/write files; secure envd |

Not production-grade (no HA, multi-tenant auth, custom templates).

## Goal

```python
s = Sandbox.create(template="base")  # secure=True
s.files.write("/tmp/x", "data")
assert s.files.read("/tmp/x") == "data"
s.commands.run("cat /tmp/x").stdout.strip() == "data"
s.kill()
```

```bash
helm install actordock ./charts/actordock-stack -n actordock --create-namespace
```

## In scope

| Area | Deliverable |
|------|-------------|
| envd | Filesystem RPC: read, write, list |
| Platform | `secure=true`; `envd_access_token` on create |
| Router | Forward auth headers to envd |
| Deploy | Helm `actordock-stack`; Substrate pin in chart/lock |
| Docs | Install runbook, env vars, troubleshooting |

## Out of scope

- Custom templates, pause/connect API
- Metrics, logs, network rules
- Multi-tenant auth, HA, SLA
- Full E2B OpenAPI parity

## API

| Component | Additions |
|-----------|-----------|
| envd | `filesystem.read`, `write`, `list` |
| Platform | Token on create when `secure=true` |
