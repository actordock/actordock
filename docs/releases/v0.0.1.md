# v0.0.1

**Status:** planned

## Target

**Proof** — A sandbox is a real workload in the cluster, not a mock.

| Before | After |
|--------|-------|
| No Actordock | End-to-end path: SDK → Platform → Substrate → envd → stdout |

## Goal

```python
from e2b import Sandbox
s = Sandbox.create(template="base", secure=False)
print(s.commands.run("echo hello").stdout)  # "hello"
s.kill()
```

```bash
export E2B_API_URL=http://localhost:8080
export E2B_SANDBOX_URL=http://localhost:8081
export E2B_DOMAIN=localhost
export E2B_API_KEY=dev
export E2B_VALIDATE_API_KEY=false
./hack/install-local.sh && python examples/verify.py
```

## In scope

| Area | Deliverable |
|------|-------------|
| Deploy | `./hack/install-local.sh` — Kind `actordock` + pinned Substrate + Actordock |
| Substrate | WorkerPool, ActorTemplate `base`, envd golden snapshot Ready |
| Platform | `GET /health`, `POST /sandboxes`, `DELETE /sandboxes/{id}` |
| Router | `{sandboxID}.localhost` → envd :49983 |
| envd | `/init`, ConnectRPC `process.Start` |
| Template | Hardcoded `base` only |

## Out of scope

- Query/list sandboxes, timeout, scheduler
- Filesystem, secure tokens, Redis
- Helm, HA, custom templates

## API

| Method | Path |
|--------|------|
| GET | `/health` |
| POST | `/sandboxes` |
| DELETE | `/sandboxes/{id}` |

Create response: `sandbox_id`, `domain`, `envd_version` (≥ 0.1.0), `envd_access_token` (empty if `secure=false`).

## Milestones

1. Kind cluster `actordock` + Substrate control plane healthy
2. ActorTemplate `base` → Ready
3. `POST /sandboxes` → actor + envd pod in cluster
4. Router → `process.Start` → stdout
5. `verify.py` green
